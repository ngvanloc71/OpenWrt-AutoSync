# OpenWrt AutoSync

Mirror two NTFS disks on OpenWrt.

## Features

- NTFS → NTFS mirror
- lsyncd + rsync
- USB hotplug
- Telegram notification
- Health monitor
- LuCI support
- Auto remount
- Auto recovery

## Status

Development.

## Router Tested

- Netgear R7800
- Linksys WRT3200ACM
